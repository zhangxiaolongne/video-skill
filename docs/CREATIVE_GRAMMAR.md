# Creative Grammar

This document is the canonical human-readable contract for reusable creative
vocabulary. The master owns long-range principles; code owns executable models;
contract tests require this vocabulary and the code to stay aligned.

The lists below are maintained seeds, not closed product limits. Host-Agent or
user-supplied project vocabulary may extend them through a future validated
boundary. An example must never become a maximum.

## Composition Model

```text
content/output form
x aesthetic style
x creative strategy
x creative technique
x emotional arc
x rule mode (follow / bend / break)
```

Source type informs content compatibility. It does not determine aesthetic
style. No combination is selected or applied without explicit user choice.

## Content Forms

- `stage_performance_portrait`: stage and live-performance person portrait
- `interview_portrait`: interview-led person portrait
- `event_montage`: public-event or variety montage
- `short_talking_head`: compact speech-led short video
- `promotional_short`: promotional person/event short
- `documentary_portrait`: evidence-led documentary portrait
- `film_role_montage`: film-role montage
- `television_role_arc`: television-role arc
- `music_video_portrait`: music-video person portrait
- `musical_theatre_portrait`: musical-theatre person portrait
- `theatre_performance_portrait`: theatre-performance portrait
- `variety_persona`: variety-show persona edit
- `rehearsal_process`: rehearsal/process edit
- `behind_scenes_portrait`: behind-the-scenes portrait
- `cross_media_portrait`: cross-media person portrait
- `fan_creation_montage`: fan-created or transformative montage

These forms cover current seeds for performance, spoken material, fiction,
process, event, promotion, documentary, cross-media work, and fan creation.
They may be mixed when source ownership and semantic context remain explicit.

## Aesthetic Styles

- `idol_gloss`: radiant, polished, subject-forward idol treatment
- `hot_blooded`: pressure, drive, impact, and explosive release
- `inspirational`: evidenced struggle, growth, return, and hope
- `restrained_premium`: controlled pacing, negative space, and aftertaste
- `cinematic`: motif, immersion, scale, sound design, and dramatic variation
- `healing`: gentleness, safety, breathing room, and renewal
- `romantic`: intimacy, longing, visual rhyme, and delayed completion
- `melancholic`: distance, loss, fragments, silence, and unresolved feeling
- `dreamy`: wonder, drift, ambiguity, loops, and temporal slippage
- `epic`: scale, destiny, decisive contrast, and earned triumph
- `nostalgic`: memory, time, texture, recurrence, and source-respectful warmth
- `documentary_raw`: labor, environment, imperfection, and evidence-first truth
- `experimental_contrast`: collision, friction, mismatch, and reinterpretation
- `absurdist_play`: humor, estrangement, deadpan holds, and intentional mismatch

Styles may be mixed by section. Style names do not prove that their emotion is
present in source material; semantic claims still require transcript, visual,
audio, contextual, user, or host-reviewed evidence.

## Creative Techniques

- `chronological_continuity` (`follow`): preserve source order when time progression carries meaning
- `nonlinear_reorder` (`bend`): reorder evidence-backed ranges without inventing causality
- `delayed_payoff` (`bend`): withhold completion or climax so release is earned later
- `extreme_reversal` (`break`): reverse style, image, sound, rhythm, or interpretive frame sharply
- `sound_image_dislocation` (`break`): separate image and sound time/meaning while preserving ownership
- `intentional_silence` (`bend`): remove or hold sound to focus attention on a real moment
- `long_take_breathing` (`break`): intentionally refuse an expected cut so performance can breathe
- `contrast_montage` (`bend`): juxtapose opposing roles, media, states, or moments with context
- `motif_repetition` (`bend`): repeat an evidenced image, gesture, phrase, or sound to accumulate meaning
- `intentional_rupture` (`break`): break established pacing, composition, or style at a real turn

Every technique, especially `break`, must retain six explanations:

1. **Form**: what image/sound/timing relation changes.
2. **Feeling**: what the viewer is expected to experience.
3. **Meaning**: why it serves this person, theme, or moment.
4. **Risk**: how it may become random, false, exploitative, or excessive.
5. **Playback verification**: what observable result proves it works.
6. **Fallback**: what conservative construction replaces it if it fails.

## Emotional Arcs

- `linear_rise`: setup -> build -> release
- `quiet_late_release`: restraint -> accumulation -> late release
- `suppression_to_explosion`: suppression -> pressure -> explosion -> aftershock
- `admiration_to_vulnerability`: radiance -> fracture -> vulnerability -> renewed regard
- `worship_distance_reframe`: idolization -> distance -> contradiction -> reinterpretation
- `joy_to_silence`: joy -> acceleration -> rupture -> silence
- `fall_and_return`: promise -> failure -> effort -> return
- `cyclical_memory`: present -> memory -> variation -> return
- `unresolved_aftertaste`: question -> fragments -> near-answer -> open end
- `comic_misdirection`: setup -> false expectation -> reveal -> deadpan hold

Arcs may be subtle, strong, or extreme. Failure, vulnerability, relationship,
struggle, recovery, and reversal must not be invented to satisfy an arc.

## Rule Modes

- `follow`: use the template's default professional convention.
- `bend`: vary the convention while preserving its purpose.
- `break`: deliberately violate the convention for an evidenced creative reason.

Rules are tools, not creative ceilings. `break` is allowed for discontinuity,
blank space, contrast, sound-image mismatch, delayed completion, deliberate
non-cutting, nonlinear order, and extreme reversal. Random novelty without the
six explanations is not valid creative freedom.

## Audiovisual Coupling

Every selected combination must coordinate:

- duration and structure;
- source audio, silence, dialogue, ambience, and BGM;
- BGM input provenance, phrase/beat availability, and ducking;
- subtitle density, entrance/exit timing, and safe regions;
- shot duration, transitions, composition, and reframing;
- emotional arc, reversal point, ending, and playback acceptance.

An extracted video mix is not automatically clean BGM. Technical energy is not
emotion. A template match is not an applied edit. A technically valid render is
not automatically aesthetically publishable.

## Extension Contract

New content forms, aesthetic styles, techniques, and arcs are allowed when they:

- use stable snake-case IDs;
- state intended effect and evidence requirements;
- retain provenance and uncertainty;
- define risks and playback verification;
- do not fabricate dialogue, lyrics, roles, relationships, events, or emotion;
- do not silently select, apply, render, or overwrite canonical evidence;
- remain compatible with one canonical JSON plus one human report unless an
  independent external boundary genuinely requires another artifact.
