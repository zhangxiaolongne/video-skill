# Vision

Authoritative source: `artist_portrait_editor_revision5_optimized.md`.

`artist-portrait-editor` is a person-centered editing assistant for broad
portrait-oriented video creation. It is not limited to idols, stage footage,
music-driven cuts, or beat matching.

Long-term goal:

```text
research materials
-> develop creative concepts
-> design narrative and sound
-> draft edits
-> support professional project output
```

V0 success is not one-click finished video. V0 should first become a reliable
material researcher and explainable editing planner.

Current implementation gate: V0-004 fixed-window segmentation foundation only.
